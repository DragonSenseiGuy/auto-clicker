Auto Clicker

A free and open-source auto clicker

About Auto Clicker

Auto Clicker is a free and open-source application designed to automate mouse clicks. It helps users perform repetitive clicking tasks with ease, offering customization options for click type, rate, and activation hotkeys.

Features

  • Support for left and right clicking
  • Configurable click rate (clicks per second)
  • Customizable hotkey for activation and deactivation
  • Intuitive Graphical User Interface (GUI)
  • Displays the current status of the auto-clicker (active/inactive)

Installation & Usage

Follow these steps to get the Auto Clicker up and running:

1. Download the Application

Download the latest version of Auto Clicker here.This build is only tested on macOS.

2. Run the application (from source)

If you prefer to run from source, first install the dependencies:

uv sync

Then, run the application:

uv python run main.py

3. How to Use

  1. Configure your settings:
    • Select click type (Left or Right)
    • Adjust clicks per second using the slider
    • Set your preferred hotkey (optional)
  2. Start auto clicking:
    • Click the "Start" button, or
    • Press your configured hotkey (default: *)
  3. Stop auto clicking:
    • Click the "Stop" button, or
    • Press your hotkey again to toggle
